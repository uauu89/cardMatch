# 카드 맞추기 게임

<br />

## 개요

### 기획의도
+ 리액트 공부를 위한 프로젝트 
  + 컴포넌트 사이의 값, 함수 전달을 학습하기 위해 Redux 라이브러리는 배제
  + css-module 사용

### 디자인컨셉
+ 외부 이미지 없이 CSS로 디자인 구현

### 주요기능
+ 컴퓨터와의 대전 기능 및 난이도 옵션 구현
+ 타이머 기능 및 제한 없음 옵션 구현
+ 반응형 페이지로 제작
+ 입력창 커스텀 디자인 적용 및 접근성을 고려하여 탭키로 접근 가능하도록 제작
+ 외부 이미지 없이 CSS로 디자인
